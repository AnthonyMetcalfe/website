Mastermind Tutorial

The point of mastermind is to guess the four colors, in order, that the creator set. Each time you make a guess, markers to the right indicate how well you did. A black marker means that you got one of the colors in the exact right location, while a white marker means that you got the right color, but put it in the wrong spot.

The classic game of mastermind doesn't allow for the same color to appear twice in a puzzle. Here, however, there is an option to play with duplicates allowed. If you are looking to play an existing game that allows duplicates, look for a hot pink tag indicating so.

To create a new game, click the plus button (duh).

If you leave the game and come back to it, the rounds already used will be filled in.
